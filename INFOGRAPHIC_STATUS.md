## 🟢 STATUS LUNAS (Hijau)

```
┌─────────────────────────────────────────────────────────────┐
│  🟢 LUNAS                                                    │
│                                                             │
│  📊 Kondisi:                                                 │
│     • Jumlah dibayar = Jumlah utang                         │
│     • Atau lebih dari jumlah utang                          │
│                                                             │
│  💰 Contoh:                                                  │
│     Utang: Rp 1.000.000                                     │
│     Dibayar: Rp 1.000.000                                   │
│     Status: 🟢 LUNAS                                         │
│                                                             │
│  ✅ Tindakan:                                                │
│     • Bisa diarsipkan                                       │
│     • Tidak perlu khawatir                                  │
│     • Hubungan baik terjaga                                 │
└─────────────────────────────────────────────────────────────┘
```

---

## 🔵 STATUS DIBAYAR SEBAGIAN (Biru)

```
┌─────────────────────────────────────────────────────────────┐
│  🔵 DIBAYAR SEBAGIAN                                         │
│                                                             │
│  📊 Kondisi:                                                 │
│     • Sudah ada pembayaran                                  │
│     • Tapi belum mencapai jumlah penuh                      │
│                                                             │
│  💰 Contoh:                                                  │
│     Utang: Rp 1.000.000                                     │
│     Dibayar: Rp 500.000                                     │
│     Sisa: Rp 500.000                                        │
│     Status: 🔵 DIBAYAR SEBAGIAN                             │
│                                                             │
│  ⚠️ Tindakan:                                                │
│     • Lanjutkan pembayaran                                  │
│     • Buat rencana pelunasan                                │
│     • Monitor sisa utang                                    │
└─────────────────────────────────────────────────────────────┘
```

---

## 🟡 STATUS BELUM DIBAYAR (Kuning)

```
┌─────────────────────────────────────────────────────────────┐
│  🟡 BELUM DIBAYAR                                            │
│                                                             │
│  📊 Kondisi:                                                 │
│     • Belum ada pembayaran sama sekali                      │
│     • Masih dalam waktu jatuh tempo                         │
│                                                             │
│  💰 Contoh:                                                  │
│     Utang: Rp 1.000.000                                     │
│     Dibayar: Rp 0                                            │
│     Sisa: Rp 1.000.000                                      │
│     Status: 🟡 BELUM DIBAYAR                                │
│                                                             │
│  ⏳ Tindakan:                                                │
│     • Perlu diperhatikan                                    │
│     • Buat rencana pembayaran                               │
│     • Siapkan dana untuk pelunasan                          │
└─────────────────────────────────────────────────────────────┘
```

## 🔴 STATUS TERLAMBAT (Merah)

```
┌─────────────────────────────────────────────────────────────┐
│  🔴 TERLAMBAT                                                │
│                                                             │
│  📊 Kondisi:                                                 │
│     • Sudah melewati tanggal jatuh tempo                    │
│     • Bisa belum dibayar atau dibayar sebagian              │
│                                                             │
│  💰 Contoh 1 - Belum Dibayar:                               │
│     Utang: Rp 1.000.000                                     │
│     Jatuh Tempo: 15 Jan 2025                                │
│     Hari Ini: 20 Jan 2025                                   │
│     Dibayar: Rp 0                                           │
│     Status: 🔴 BELUM DIBAYAR (Terlambat 5 hari)            │
│                                                             │
│  💰 Contoh 2 - Dibayar Sebagian:                            │
│     Utang: Rp 1.000.000                                     │
│     Jatuh Tempo: 15 Jan 2025                                │
│     Hari Ini: 20 Jan 2025                                   │
│     Dibayar: Rp 500.000                                     │
│     Status: 🔴 DIBAYAR SEBAGIAN (Terlambat 5 hari)         │
│                                                             │
│  🚨 Tindakan:                                                │
│     • SEGERA tindaklanjuti                                  │
│     • Hubungi kreditur/debitur                              │
│     • Buat kesepakatan pembayaran                           │
└─────────────────────────────────────────────────────────────┘
```
